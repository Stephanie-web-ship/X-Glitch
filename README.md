# X-GLITCH: Exoskeleton Gauntlet Linked Interface for Therapy, Control, and Healing

# Soft Wearable System for Adaptive Gamified Hand Rehabilitation Robotics

# Overview

X-GLITCH is an AI-enabled wearable hand rehabilitation exoskeleton framework designed to assist individuals with hand mobility impairments caused by stroke, neurological disorders, injuries, or rehabilitation requirements.

The system aims to overcome the limitations of conventional repetitive therapy by combining wearable robotics, real-time monitoring, adaptive intelligence, and gamified rehabilitation. X-GLITCH provides personalized therapy assistance while allowing patients and healthcare professionals to track rehabilitation progress.

# Problem Statement

Traditional hand rehabilitation methods often involve repetitive exercises that require continuous clinical supervision. These exercises may become monotonous, reducing patient motivation and long-term therapy adherence.

Existing rehabilitation systems may lack:
- Personalized therapy adaptation
- Finger-specific progress monitoring
- Engaging rehabilitation approaches
- Continuous performance tracking

X-GLITCH addresses these challenges through an intelligent closed-loop rehabilitation framework.

# Proposed Solution

The proposed system consists of a lightweight wearable exoskeleton integrated with:

- Tendon-driven actuation mechanism
- Embedded sensing system
- Wireless communication
- Adaptive rehabilitation intelligence
- Interactive gaming-based therapy platform

The system monitors patient performance and adjusts rehabilitation difficulty based on individual progress.

# Key Features
# Lightweight Exoskeleton Design
- Carbon nanofiber-based structure
- Dorsal exoskeleton architecture
- Reduced hand coverage compared to glove-based systems
- Improved comfort and wearability

# Tendon-Driven Actuation
- Wrist-mounted BLDC motors
- Pulley and tendon transmission system
- Assisted finger flexion and extension

# Embedded Hardware System
- STM32 microcontroller
- Hall effect sensors for finger movement detection
- IMU sensor for hand motion tracking
- Force and current monitoring
- Rechargeable lithium-ion battery system
- Bluetooth Low Energy (BLE) connectivity

# Rehabilitation Modes

# Passive Mode
The exoskeleton performs complete finger movement assistance for patients with limited voluntary hand mobility.

# Active-Assisted Mode
The patient initiates movement while the exoskeleton provides assistance when required to complete the motion.

# Software Framework

The software layer consists of a mobile-based rehabilitation platform that provides:

- Real-time progress monitoring
- Therapy session tracking
- Gamified rehabilitation exercises
- Patient performance dashboard
- Doctor monitoring interface

# AI-Based Adaptive Rehabilitation

The intelligence layer analyzes:

- Range of motion
- Finger movement accuracy
- Exercise completion rate
- Recovery patterns

Based on patient progress, the system dynamically adjusts:
- Therapy difficulty
- Game challenges
- Finger-specific rehabilitation tasks

# System Architecture

The X-GLITCH framework consists of three layers:

1. Hardware Layer
   - Exoskeleton
   - Sensors
   - Actuators
   - Embedded controller

2. Software Layer
   - Mobile application
   - Gamified therapy interface
   - Progress dashboard

3. Intelligence Layer
   - Performance analysis
   - Adaptive therapy generation
